## :computer: Como rodar o projeto

1. `$ git clone https://github.com/MichelSchiavo/TractianFrontV2` para clonar o projeto
2. `$ yarn ` para instalar as dependências do projeto
3. `$ yarn start` para iniciar o servidor local
